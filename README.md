# Yunming OS  

## 简介

这是一个基于arceos的unikernel操作系统。这里还提供了最基本的命令行界面，以及基本的文件操作：

![](./assets/help.png)

目前支持的架构如下:

- [x] x86_64
- [x] ARM架构
- [ ] RISC-V
- [ ] LoongArch

如果你想快速体验，你可以跳转到这个项目[](https://github.com/HugoPhi/yunmingos.git)，这里有编译好的内核文件，以及详细的指导教程，让你能够在Docker里面体验这个unikernel操作系统。
